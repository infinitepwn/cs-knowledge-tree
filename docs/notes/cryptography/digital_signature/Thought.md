# Thought

> 来源：`密码学引论 / digital_signature/Thought.md`

在AI逐渐发展的今天，如何评估一个知识是否学会是一件困难的事情

记住某个概念？会使用这个知识解决一些问题？但如果这个知识是一个零碎的片段，好像并没有什么作用，因为我们可以通过AI迅速检索

所以核心是一个框架？我们需要掌握的是一个思维运行的逻辑链，我们顺着这个逻辑链，可以分析这个问题，即使我们不知道或者暂时忘记了某个概念，也可以通过笔记或者AI检索迅速学会，从而使得这个逻辑链条继续运行？体系化的思维是非常重要的

AI的一大优势就是可以迅速答疑，所以此处的学习笔记可以以QA的形式来呈现
