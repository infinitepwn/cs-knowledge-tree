# 批处理系统

> 来源：`操作系统 / note/概述/批处理系统.md`

上级：[操作系统概述](操作系统概述.md)

批处理系统把一批作业集中提交给计算机，由操作系统按顺序或按调度策略自动运行，减少人工干预。

## 特点

- 用户不直接和正在运行的程序交互。
- 系统吞吐量是重要目标。
- 作业等待时间可能较长。

## 分类

- 单道批处理：内存中一次只运行一个作业。
- 多道批处理：内存中同时保留多个作业，通过[调度](../进程管理/调度/调度.md)提高 CPU 利用率。
