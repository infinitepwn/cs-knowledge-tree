# i节点

> 来源：`操作系统 / note/文件管理/i节点.md`

i节点是文件系统中用于描述文件的数据结构，保存文件的属性以及文件数据块的位置。

## 作用

i节点用于把文件的元数据和文件名分离。

文件名通常保存在 [目录](目录.md) 中，而文件的权限、大小、时间、磁盘块地址等信息保存在 [i节点](i节点.md) 中。

## 和 FAT 的区别

[FAT](FAT.md) 需要把整张文件分配表放在内存中。

[i节点](i节点.md) 只需要在文件被打开时加载对应的节点，因此更适合大型磁盘。
