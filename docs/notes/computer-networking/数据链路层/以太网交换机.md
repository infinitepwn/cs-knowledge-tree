# 以太网交换机

> 来源：`计算机网络 / 数据链路层/以太网交换机.md`

## 基本概念

- 交换机本质上是多端口网桥
- 交换机的转发和过滤
- 独占带宽
- 全双工通信

## 碰撞域和广播域

路由器可以分隔广播域，交换机可以分隔碰撞域

![image.png](https://raw.githubusercontent.com/infinitepwn/note_picbed/main/20260615143008160.png)

## 工作过程

- 看源 MAC 地址：学习该地址所在端口，更新交换表。
- 看目的 MAC 地址：如果交换表中有记录，则定向转发。
- 如果目的地址未知，则向除进入端口以外的所有端口泛洪。
- 如果目的端口就是进入端口，则过滤该帧。
