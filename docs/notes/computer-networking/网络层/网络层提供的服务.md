# 网络层提供的服务

> 来源：`计算机网络 / 网络层/网络层提供的服务.md`

## 虚电路服务
认为应当让网络层负责可靠交付，在分组交换中建一条虚电路，双方沿着已建立的虚电路发送分组

虚电路服务采用分组交换，具有存储转发机制

只有建立连接的分组需要携带目的地址

有差错控制和流量控制
## 数据报服务
数据报=IP数据报=IP分组=分组
简便灵活、无连接的、尽最大努力交付的数据报服务
- 互联网为什么采用无连接的数据报服务
- 网络层是否保证可靠交付
