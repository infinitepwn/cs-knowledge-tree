# BGP协议

> 来源：`计算机网络 / 网络层/BGP协议.md`

## 定义

BGP（Border Gateway Protocol，边界网关协议）是外部网关协议，用于自治系统 AS 之间的路由选择。

BGP 使用 [TCP协议](../运输层/TCP协议.md)，端口号 179，用 TCP 连接保证路由信息交换的可靠性。

## 特点

- BGP 用于自治系统 AS 之间。
- BGP 是路径向量路由选择协议。
- BGP 的可达信息是路径，而不只是距离。
- BGP 尽量寻找无环路路径。

## 与 RIP、OSPF 的区别

- [RIP协议](RIP协议.md) 和 [OSPF协议](OSPF协议.md) 主要用于自治系统内部。
- BGP 用于自治系统之间，更关注策略和可达性，而不是单纯寻找最短路径。
