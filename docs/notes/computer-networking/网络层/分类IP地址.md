# 分类IP地址

> 来源：`计算机网络 / 网络层/分类IP地址.md`

## A 类地址

网络号8位
第一位是0
127开头的都是回环地址，保留地址，不可用

## B 类地址

网络号16位
前两位是10

## C 类地址

前三位是110
网络号24位
只有$2^8-2=254$个主机可以使用
ABC类都是单播地址（全1是广播地址）
192开头的肯定是C类

## D 类地址与 E 类地址

D类地址是多播地址
1110开头，也就是224开头
特殊地址

### 私有地址

- **A 类**：`10.0.0.0` ～ `10.255.255.255`
- **B 类**：`172.16.0.0` ～ `172.31.255.255`
- **C 类**：`192.168.0.0` ～ `192.168.255.255`（例如常见的路由器网关 `192.168.1.1`）

## 分类地址的局限

分类地址的网络规模固定，地址分配不够灵活，容易造成地址浪费。因此后来引入 [CIDR](CIDR.md)，用网络前缀长度按需划分地址块。
