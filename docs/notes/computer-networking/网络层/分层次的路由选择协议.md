# 分层次的路由选择协议

> 来源：`计算机网络 / 网络层/分层次的路由选择协议.md`

## 定义

互联网采用分层的路由选择协议，把一整个互联网分成许多小的自治系统(Autonomous System),AS

![image.png](https://raw.githubusercontent.com/infinitepwn/note_picbed/main/20260615211420601.png)

## 内部网关协议IGP

在自治系统内部使用的协议，如[RIP协议](RIP协议.md)、[OSPF](OSPF协议.md)

## 外部网关协议EGP

在不同的自治系统之间使用的协议，如[BGP协议](BGP协议.md)
