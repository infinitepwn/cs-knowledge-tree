# 分组交换

> 来源：`计算机网络 / 概述/分组交换.md`

分组交换采用存储转发技术

我们把要发送的整个数据叫做报文(message)

![image.png](https://raw.githubusercontent.com/infinitepwn/note_picbed/main/20260605200242791.png)

将较长的报文划分成一个个小的等长数据段，再加上一些必要的控制信息组成的首部(header)之后，就构成了一个分组(packet)，分组又被称为包，首部叫做包头

分组是互联网中传送的数据单元

单个分组传送到相邻节点，存储下来后查找转发表，转发到下一个节点

分组交换类似流水线，比报文交换快

设分组长度为Lbit，传输速度为Rbps,则分组交换可以类比成**流水线**，如果有P个分组需要经过N条链路（N-1个路由器）

那么总时间为


$$
t = (N+P-1)\frac{L}{R}
$$
