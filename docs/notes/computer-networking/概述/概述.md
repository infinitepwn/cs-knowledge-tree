# 概述

> 来源：`计算机网络 / 概述/概述.md`

计算机网络(网络)由若干个节点和链路组成
节点可以是计算机、集线器、交换机或路由器等

互连网（internet）
多个网络通过一些路由器相互连接起来，形成一个更大的网络，也被称为网络的网络
![image.png](https://raw.githubusercontent.com/infinitepwn/note_picbed/main/20260605193248515.png)

- **网络：把许多计算机连接在一起。
- **互连网**：把许多网络通过一些路由器连接在一起。与网络相连的计算机常称为主机。
- **互联网**：指当前全球最大的、开放的、由众多网络互相连接而成的特定的互连网，他采用TCP/IP协议族作为通信规则，且前身为美国的ARPANET。
- **互连网** **(internet) ≠** **互联网** **(Internet)**

互联网是最大的互连网

**⽹络把许多计算机连接在⼀起,⽽互连⽹则把许多⽹络通过⼀些路由器连接在⼀起｡与⽹络相连的计算机常称为主机｡**

## [互联网的组成](互联网的组成.md)

从互联网的工作方式，可以分成两个部分
### [互联网的边缘部分](互联网的边缘部分.md)
互联网的**边缘部分**主要是各种端系统，比如手机、电脑、服务器。它们负责运行应用程序，产生和接收数据。边缘部分常见的[通信方式](通信方式.md)有[客户-服务器模式](客户-服务器模式.md)和 [P2P 模式](P2P 模式.md)。
### [互联网的核心部分](互联网的核心部分.md)
互联网的**核心部分**主要由[路由器](路由器.md)、交换机、通信链路和运营商网络组成。它的主要任务是把分组从源主机转发到目的主机。互联网核心主要采用[分组交换](分组交换.md)技术，通过存储转发、路由选择和排队机制完成数据传输。
