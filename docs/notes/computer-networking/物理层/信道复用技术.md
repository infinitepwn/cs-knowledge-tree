# 信道复用技术

> 来源：`计算机网络 / 物理层/信道复用技术.md`

## 什么是信道复用？

![image.png|379](https://raw.githubusercontent.com/infinitepwn/note_picbed/main/20260610223807519.png)

信道复用技术可以让多个主机或多路信号共享同一条信道，提高通信线路的利用率。

基本思想是在发送端使用复用器，把多路信号合并到同一条共享信道中传输；在接收端使用分用器，把合并后的信号重新分离出来。

## 常见复用方式

| 复用方式 | 英文缩写 | 核心思想 | 典型特点 |
| --- | --- | --- | --- |
| [频分复用](频分复用.md) | FDM | 不同用户占用不同频带 | 同一时间传输，频率不同 |
| [时分复用](时分复用.md) | TDM | 不同用户占用固定时隙 | 时间划分固定，适合等时业务 |
| 统计时分复用 | STDM | 按需动态分配时隙 | 提高线路利用率，适合突发数据 |
| [波分复用](波分复用.md) | WDM | 不同光信号使用不同波长 | 本质上是光的频分复用 |
| [码分复用](码分复用.md) | CDM | 不同用户使用不同码片序列 | 同时同频传输，依靠码型区分 |

## 对比理解

- 频分复用：把各路信号搬移到不同的频率位置。
- 时分复用：把时间划分为周期性的时隙，各路信号轮流使用信道。
- 波分复用：用于光纤通信，把不同波长的光信号合在同一根光纤中传输。
- 码分复用：各站在同一时间、同一频带内通信，通过正交码片序列区分用户。

## 易混点

- TDM 的固定时隙即使某用户暂时没有数据，也可能被空置。
- STDM 不再给每个用户固定时隙，而是根据实际数据发送需求动态分配。
- WDM 常说成“光的频分复用”，因为波长和频率本质相关。
- CDMA 是码分多址，是码分复用在多用户接入场景中的应用。
