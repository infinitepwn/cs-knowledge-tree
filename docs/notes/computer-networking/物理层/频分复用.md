# 频分复用

> 来源：`计算机网络 / 物理层/频分复用.md`

## 定义

![image.png](https://raw.githubusercontent.com/infinitepwn/note_picbed/main/20260610224100448.png)

频分复用（Frequency Division Multiplexing，FDM）通过调制的方法，把 $N$ 路信号搬移到不同的频率范围，使它们在同一条信道中同时传输而互不干扰。

## 特点

- 各路信号在同一时间传输。
- 各路信号占用不同的频带资源。
- 为避免相邻频带互相干扰，通常需要留出保护频带。
