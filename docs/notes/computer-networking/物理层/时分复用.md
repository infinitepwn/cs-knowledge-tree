# 时分复用

> 来源：`计算机网络 / 物理层/时分复用.md`

## 定义

时分复用（Time Division Multiplexing，TDM）把时间划分成一段段等长的时分复用帧（TDM 帧），每一路信号在每个 TDM 帧中占用固定序号的时隙。

![image.png](https://raw.githubusercontent.com/infinitepwn/note_picbed/main/20260610224357520.png)

## 特点

- 各路信号在不同时间占用同样的频带宽度。
- 时隙分配固定，适合周期性、等时的数据流。
- 如果某一路暂时没有数据，它对应的固定时隙也可能被浪费。

## 与统计时分复用的区别

统计时分复用（STDM）不固定给每一路信号分配时隙，而是根据实际是否有数据来动态分配时隙，因此更适合突发式数据通信。
