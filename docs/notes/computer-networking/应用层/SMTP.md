# SMTP

> 来源：`计算机网络 / 应用层/SMTP.md`

## 定义

SMTP（Simple Mail Transfer Protocol，简单邮件传送协议）用于发送邮件。

SMTP 使用 [TCP协议](../运输层/TCP协议.md)，是“推”协议：发送方主动把邮件推送到接收方邮件服务器。

## 与读取邮件协议的区别

- SMTP：用于发送邮件。
- [POP3](POP3.md)：用于从邮件服务器读取邮件，通常偏向下载到本地。
- [IMAP](IMAP.md)：用于从邮件服务器读取和管理邮件，适合多设备同步。
