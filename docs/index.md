# CS Knowledge Tree

这是一个类似课程 Wiki 的计算机科学知识站。它把你的课程笔记按知识域和技术栈重新组织，而不是只按课程目录堆放。

## 快速入口

<div class="grid cards" markdown>

-   **学习路线图**

    从系统基础、网络基础、密码学与安全一路到工程整合。

    [进入路线图](roadmap.md)

-   **计算机系统**

    程序运行、操作系统、进程线程、虚拟内存、文件系统和 I/O。

    [进入知识树](knowledge/computer-systems.md)

-   **计算机网络**

    分层协议、IP、TCP/UDP、DNS、HTTP/HTTPS 和网络实验。

    [进入知识树](knowledge/networking.md)

-   **密码学与安全**

    数学基础、公钥密码、哈希、数字签名、密钥交换、零知识证明。

    [进入知识树](knowledge/cybersecurity.md)

## 组织原则

- 课程仓库保留完整笔记。
- 本站负责组织知识关系、学习路线和技术栈。
- 后续新增课程时，先进入课程仓库，再抽象到知识树。
